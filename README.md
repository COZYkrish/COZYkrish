## ◈ ORIGIN STORY

<div align="center">

> *"I didn't start with a computer science degree — I started with curiosity."*

</div>

It began with a simple question: *"How does a website actually work?"*

That question opened a rabbit hole that never closed. From writing my first `Hello World` in Python to designing multi-layered full-stack systems with real-time analytics, authentication flows, and cloud deployment pipelines — every line of code has been a step deeper into an obsession.

As a **BCA student at Centurion University**, I've been on a mission to go beyond the curriculum. While others study theory, I build systems. I've shipped **10+ full-stack projects** spanning fintech dashboards, AI desktop assistants, enterprise complaint platforms, and fitness analytics engines — each one a piece of a larger engineering identity.

My passion lives at the intersection of **engineering depth** and **design beauty** — where scalable backend architecture meets immersive, futuristic user interfaces. I believe software should not just *work* — it should feel *alive*.

Currently, I'm going deep on **cloud computing** and **AI integration**, learning how to architect systems that aren't just functional today, but scalable for what comes next.

**I don't just write code. I engineer experiences.**
